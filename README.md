# pygallery
Galería en la nube donde puedes subir tus vídeos y fotos.
Esta consta principalmente de 2 aplicaciones:
## pygallery (Server):
- El servidor.
- Donde se va a almacenar todo.
## pygallery Desktop Sync
- Aplicación de sincronización (inicias sesión y eliges las carpetas que quieres que se sincronizen con tu cuenta, se queda en segundo plano, Windows, Linux)
- La aplicación comprueba cada 5 minutos si algo ha sido añadido (si se quiere borrar algo, se borra desde la aplicación web)
